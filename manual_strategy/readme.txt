
How to run the code for Project 6

1. In terminal, navigate to the folder that contains all the python codes needed (i.e. indicators.py, TheoreticallyOptimalStrategy.py, ManualStrategy.py, marketsimcode.py, and util.py).

2. Input 'python indicators.py' to generate Figures 1-4 in the report.

3. Input 'python TheoreticallyOptimalStrategy.py' to generate Figure 5 in the report. Doing so will also print out the result shown in Part 2 in the terminal.

4. Input 'python ManualStrategy.py' to generate Figures 6-7 in the report. Doing so will also print out the result shown in Part 3 and Part 4 in the terminal.
